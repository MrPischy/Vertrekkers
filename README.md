# Vertrekkers Wiki

Een verzameling nuttige informatie voor mensen die met een zeilboot de wereld rond gaan. Deze informatie is verzameld uit WhatsApp groepen voor vertrekkers.

## Inhoud

- Verzekeringen
- Bankzaken
- Formaliteiten
- Huisverhuur
